# 高中物理动态演示台：教材全六册 + 力学专题实验库

这是一个面向物理老师的课堂动态演示网站。当前版本按人教版高中物理必修第一册、必修第二册、必修第三册、选择性必修第一册、选择性必修第二册、选择性必修第三册（2019 版）目录组织，覆盖 27 章 125 节，并额外加入 6 个力学专题综合实验。

## 已实现章节

### 力学专题实验库

1. 力的合成、分解与平衡专题：双力合成、正交分解、三力平衡、绳结平衡。
2. 动量守恒模型组：碰撞、爆炸、火箭、人船、弹簧小球。
3. 斜面运动专题：运动实景、受力分析、能量视角。
4. 机械能守恒与耗散专题：过山车、单摆、自由落体、弹簧发射。
5. 牛顿摆专题：撞入球数量、初始角度、耗散和冲量传递。
6. 万有引力与轨道专题：椭圆轨道、逃逸、转移轨道、双星模型。

### 必修第一册

1. 第一章 运动的描述：质点参考系、时间位移、速度、加速度。
2. 第二章 匀变速直线运动的研究：纸带实验、v-t 关系、位移公式、自由落体。
3. 第三章 相互作用——力：重力与弹力、摩擦力、牛顿第三定律、力的合成和分解、共点力平衡。
4. 第四章 运动和力的关系：牛顿第一定律、a-F/a-1/m 实验、牛顿第二定律、力学单位制、运动定律应用、超重和失重。

### 必修第二册

1. 第五章 抛体运动：曲线运动、运动的合成与分解、平抛实验、抛体运动规律。
2. 第六章 圆周运动：圆周运动、向心力、向心加速度、生活中的圆周运动。
3. 第七章 万有引力与宇宙航行：行星运动、万有引力定律、万有引力理论成就、宇宙航行、相对论时空观与牛顿力学局限性。
4. 第八章 机械能守恒定律：功与功率、重力势能、动能和动能定理、机械能守恒、验证机械能守恒实验。

### 必修第三册

1. 第九章 静电场及其应用：电荷、库仑定律、电场和电场强度、静电的防止与利用。
2. 第十章 静电场中的能量：电势能和电势、电势差、U=Ed、电容器、带电粒子在电场中的运动。
3. 第十一章 电路及其应用：电源和电流、导体电阻、电阻率测量、串并联电路、多用电表。
4. 第十二章 电能 能量守恒定律：电路中的能量转化、闭合电路欧姆定律、电池电动势和内阻测量、能源与可持续发展。
5. 第十三章 电磁感应与电磁波初步：磁场和磁感线、磁感应强度和磁通量、电磁感应、电磁波、能量量子化。

### 选择性必修第一册

1. 第一章 动量守恒定律：动量、动量定理、动量守恒定律、验证动量守恒实验、碰撞、反冲和火箭。
2. 第二章 机械振动：简谐运动、简谐运动的描述、回复力和能量、单摆、单摆测 g、受迫振动和共振。
3. 第三章 机械波：波的形成、波的描述、反射折射衍射、干涉、多普勒效应。
4. 第四章 光：折射、全反射、干涉、双缝测波长、衍射、偏振和激光。

### 选择性必修第二册

1. 第一章 安培力与洛伦兹力：安培力、洛伦兹力、带电粒子在磁场中的运动、质谱仪与回旋加速器。
2. 第二章 电磁感应：楞次定律、法拉第电磁感应定律、涡流/电磁阻尼/电磁驱动、互感和自感。
3. 第三章 交变电流：交变电流、交变电流的描述、变压器、电能的输送。
4. 第四章 电磁振荡与电磁波：电磁振荡、电磁场与电磁波、无线电波的发射和接收、电磁波谱。
5. 第五章 传感器：认识传感器、常见传感器原理及应用、传感器自动控制装置。

### 选择性必修第三册

1. 第一章 分子动理论：分子动理论基本内容、油膜法估测分子大小、分子速率分布、分子动能和分子势能。
2. 第二章 气体、固体和液体：温度和温标、气体等温变化、等压/等容变化、固体、液体。
3. 第三章 热力学定律：功热与内能改变、热力学第一定律、能量守恒定律、热力学第二定律。
4. 第四章 原子结构和波粒二象性：黑体辐射、光电效应、核式结构、氢原子光谱和玻尔模型、粒子的波动性。
5. 第五章 原子核：原子核组成、放射性衰变、核力与结合能、核裂变与核聚变、“基本”粒子。

## 运行方式

直接双击 `index.html` 可以运行。若要用本地服务预览：

```bash
python3 -m http.server 4173 --directory mechanical-waves-lab
```

然后访问 `http://localhost:4173`。
