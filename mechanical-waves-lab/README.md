# 高中物理动态演示台：教材全六册 + 章节专题演示

这是一个面向物理老师的课堂动态演示网站。当前版本不再按具体小节组织导航，而是按教材册和章标题组织，章内再切换多个演示主题。这样可以适配不同版本教材目录差异，同时把专题增强内容并入对应教材章节。

## 已实现章节

## 运行方式

直接双击 `index.html` 可以运行。若要用本地服务预览：

```bash
python3 -m http.server 4173 --directory mechanical-waves-lab
```

然后访问 `http://localhost:4173`。
