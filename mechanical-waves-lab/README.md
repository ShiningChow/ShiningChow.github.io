# 高中物理动态演示台：教材全六册 + 力学专题实验库

这是一个面向物理老师的课堂动态演示网站。当前版本不再按具体小节组织导航，而是按教材册和章标题组织，章内再切换多个演示主题。这样可以适配不同版本教材目录差异，同时保留 27 章的核心内容和力学专题实验库。

## 已实现章节

### 力学专题实验库

1. 力的合成、分解与平衡专题：双力合成、正交分解、三力平衡、绳结平衡。
2. 动量守恒模型组：一维碰撞、二维碰撞、爆炸、火箭、人船、双小球弹簧、动飞摆、小球凹槽、小球斜面。
3. 斜面运动专题：运动实景、受力分析、能量视角。
4. 机械能守恒与耗散专题：过山车、单摆、自由落体、弹簧发射。
5. 牛顿摆专题：撞入球数量、初始角度、耗散和冲量传递。
6. 万有引力与轨道专题：椭圆轨道、逃逸、转移轨道、双星模型。

## 运行方式

直接双击 `index.html` 可以运行。若要用本地服务预览：

```bash
python3 -m http.server 4173 --directory mechanical-waves-lab
```

然后访问 `http://localhost:4173`。
